# TaskTracker
Простой менеджер задач на Java (консольный). Позволяет добавлять, просматривать и отмечать задачи как выполненные, с сохранением в файл.

## Как запустить
1. Скомпилируйте:
javac src/*.java

2. Запустите:
java src.Main

## Функционал
- Добавление задач
- Список всех задач
- Завершение задачи
- Сохранение в JSON
